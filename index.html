🤖 富民系统小助手

富民催收系统悬浮工具面板

v20260714A
1

安装 Tampermonkey 扩展

在 Chrome 浏览器中安装 Tampermonkey 油猴扩展

2

创建新脚本

点击 Tampermonkey 图标 → 添加新脚本 → 将脚本代码粘贴进去 → 保存

3

访问富民系统

打开 fmcs.fbank.com，悬浮面板会自动出现在页面右上角

4

设置 Token

点击面板中的"设置"按钮，输入你的 Token（可从浏览器 Cookie 中获取）

下载脚本文件

功能列表

  • 查询短信数据（已上线）
  • 批量查询销售（即将上线）
  • 发送系统短信（即将上线）
  • 查询还款状态（即将上线）
  • 批量添加催记（即将上线）
  • 查询客户画像（即将上线）
提示：Token 可从浏览器开发者工具 → Application → Cookies 中找到名为 token 的值。脚本会自动尝试从 Cookie 中读取 Token。
