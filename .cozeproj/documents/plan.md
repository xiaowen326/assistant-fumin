# 富民系统小助手 - 开发计划

## 概述

基于易鑫系统小助手的架构，为富民催收系统（fmcs.fbank.com）开发一个 Tampermonkey 油猴脚本悬浮工具面板。首期实现**短信数据查询**功能：从案件列表中提取 userId，批量查询联系人信息，整合输出完整的短信发送数据。后续功能（批量查询、催记、外呼等）按用户提供的接口陆续迭代。

## 技术方案

| 维度 | 选择 | 理由 |
|------|------|------|
| 技术形态 | Tampermonkey 油猴脚本 | 注入到富民系统页面，与易鑫小助手一致 |
| 语言 | 原生 JavaScript（ES6+） | 油猴脚本标准，无构建依赖 |
| UI 风格 | 深色悬浮面板 + 绿色辉光 | 沿用易鑫小助手的设计，用户已熟悉 |
| Token 获取 | 从页面 Cookie 或 localStorage 读取 | 富民系统 Token 在 Header 中传递，需从登录态获取 |
| 并发策略 | Promise.all 并发查询联系人 | 与易鑫一致，提升批量查询效率 |
| 项目管理 | 工作区内直接开发 .user.js 文件 | 单文件脚本，无需复杂构建 |

## 功能模块（首期）

### 短信数据查询

**流程**：
1. 调用 `POST /cs2/user/edge/case/pagingCase` 获取案件列表（支持分页遍历全部数据）
2. 从返回的 `result.content[]` 提取每条记录的 `id`（caseId）、`userId`
3. 对每个案件并发调用 `POST /cs2/user/contact/queryContactList`（参数：`{ caseId, assetId: 209, userId }`）
4. 整合输出：案件信息 + 联系人列表（姓名、关系、手机号、城市等）

**接口详情**：

| 接口 | 方法 | 关键 Header | 参数 | 关键返回字段 |
|------|------|------------|------|-------------|
| `/cs2/user/edge/case/pagingCase` | POST | `token`, `appid: "3"` | `{ pageSize, pageNum }` | `result.content[].id`, `userId`, `userRealName`, `ownerName`, `busiName`, `totalOverdue`, `maxOverdueDays`, `statusNew` |
| `/cs2/user/contact/queryContactList` | POST | `token`, `appid: "3"` | `{ caseId, assetId: 209, userId }` | `result[].realName`, `relation`, `mobile`, `encryptMobile`, `city`, `canCall` |

**输出格式**：表格展示，支持筛选、排序、分页、复制、导出

### 公共基础设施（复用易鑫架构）

- Token 管理（自动读取 + 手动设置）
- 进度条组件（支持最小化到后台）
- 结果展示组件（表格 + 筛选 + 复制）
- 通知组件
- 弹窗组件
- 悬浮面板 UI（拖拽、折叠、缩放、精简模式）

## 实施步骤

### 阶段一：原型设计

加载 design-canvas 技能，设计富民系统小助手的悬浮面板原型页面。UI 风格沿用易鑫小助手的深色主题 + 绿色辉光，功能按钮根据首期需求调整。原型完成后提示用户确认。

### 阶段二：代码开发

1. **搭建脚本框架** — 创建 .user.js 文件，编写油猴元数据头（@match fmcs.fbank.com/*）、GM API 桥接、全局配置（BASE_URL、TOKEN、appid）、悬浮面板 HTML/CSS 骨架。涉及文件：`fumin-helper.user.js`

2. **实现短信数据查询核心逻辑** — 实现 pagingCase 分页遍历 + queryContactList 并发查询 + 结果整合。涉及文件：`fumin-helper.user.js`

3. **实现结果展示与交互** — 结果表格渲染、筛选排序、复制导出、进度条、通知弹窗。涉及文件：`fumin-helper.user.js`

4. **实现悬浮面板 UI 交互** — 拖拽、折叠、缩放、精简模式、Token 状态指示。涉及文件：`fumin-helper.user.js`

5. **验证与调试** — 在富民系统页面上安装测试，验证接口调用和 UI 交互。

## 页面规格

##### @page(/) 悬浮工具面板

**核心职责**：作为注入到富民系统页面的悬浮工具面板，提供短信数据查询等批量操作功能。
**访问路径**：注入到 fmcs.fbank.com 所有页面，右下角悬浮显示。
**布局**：
- 顶部：标题栏（"富民系统小助手"）+ 折叠按钮 + 缩放按钮 + Token 状态指示器
- 中部：功能按钮列表（首期：短信数据查询）
- 底部：励志名言随机展示
- 弹窗层：进度条弹窗、结果展示弹窗、输入弹窗
